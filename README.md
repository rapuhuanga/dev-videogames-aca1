# Breakout CUN

Proyecto esqueleto para el ACA 1 de Programación de Videojuegos - CUN.

## Requisitos
- LÖVE 2D (https://love2d.org/)
- Lua (incluido con LÖVE)

## Estructura del Proyecto
Breakout_CUN/
├── conf.lua # Configuración de la ventana
├── main.lua # Punto de entrada
└── class/ # Todas las clases del juego
├── Game.lua
├── Paddle.lua
├── Ball.lua
├── Brick.lua
├── Level.lua
└── UI.lua
